# Homebound
Homebound is a cozy and humorous indie game set in Singapore that invites players to experience the everyday routine of coming home from a different perspective. By transforming ordinary household moments into interactive adventures, Homebound encourages players to slow down, look around and discover the humour, warmth and hidden magic that often go unnoticed in the rush of daily life.

# Features
- **Minigames**: A collection of fun minigames inspired by everyday moments at home.
- **Day-Night Cycle**: An in-game time system that makes the experience more immersive.
- **Rewards System**: Earn coins and spend them on room decor items in the shop.

# Getting Started
## Installation
1. **Clone the repository**: Clone this repository to your local machine.

   ```shell
   git clone https://github.com/leejh-23/Pygame_26
   ```

2. **Navigate to the repository**: Change to the repository's directory.

   ```shell
   cd your-repository-name
   ```

3. **Install dependencies**: Install the necessary dependencies by running the following command.

   ```shell
   pip install -r requirements.txt
   ```

4. **Run the game**
    ```shell
    py homebound.py
   ```

## Usage
1. Click the 'Play' button on the home page to start the game.
2. Have fun exploring and playing the minigames!
